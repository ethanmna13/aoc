# README

## Setup

**Prerequisites:**

Ruby
Rails
Node
NPM / Yarn

Install backend dependencies:
`bundle install`

Run migrations:
`rails db:migrate`

Create User Account
`rails db:seed`

Admin email: `spongebob@gmail.com`
Admin pw: `1234abcd`

Run api
`rails s`

Install frontend dependencies:
`cd frontend`
`npm install`

Run frontend server:
`npm run dev`
